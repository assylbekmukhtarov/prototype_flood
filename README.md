# Мониторинг снега и паводков

Прототип мониторинга и прогнозирования паводков, вызванных таянием снега.  
Работает без GEE и без токенов. Полностью на фронтенде — бэкенд не нужен.

Данные Sentinel-2 читаются напрямую из AWS S3 через STAC-каталог Element84 и библиотеку geotiff.js.

## Запуск

### Локально (Python)

```bash
python -m http.server 8000
```

Открыть: http://localhost:8000

### Docker

```bash
docker compose up --build
```

Открыть: http://localhost:8080

### GitHub Pages

Настройки репо → Pages → Source: **Deploy from branch** → `main` / `/ (root)`

## Как пользоваться

1. Нарисовать прямоугольник или полигон на карте
2. Задать даты и пороги в боковой панели
3. Температура подтягивается автоматически из Open-Meteo
4. Нажать **«Анализировать»** — ждать 5–60 секунд
5. На карте появятся три слоя: снег (синий), вода (голубой), новое затопление (красный)
6. Слои включаются/выключаются чекбоксами в легенде
7. **«Показать RGB-снимок»** — накладывает реальный снимок Sentinel-2
8. **«Скачать CSV»** — экспорт метрик

## Структура

```
prototype_flood/
├── index.html           # точка входа
├── Dockerfile
├── docker-compose.yml
└── static/
    ├── css/style.css
    └── js/
        ├── stac.js      # поиск снимков через Element84 STAC API
        ├── bands.js     # чтение COG из AWS S3 (geotiff.js + proj4js)
        ├── indices.js   # NDSI, MNDWI
        ├── masks.js     # бинарные маски + фильтрация шума
        ├── area.js      # площадь в км²
        ├── flood.js     # детекция новых затоплений
        ├── risk.js      # весовая модель оценки риска
        ├── temperature.js # температура из Open-Meteo
        ├── history.js   # исторический профиль за N лет
        ├── render.js    # отрисовка масок на Leaflet
        └── app.js       # оркестрация, карта, UI
```

## Оценка риска

Весовая модель из четырёх факторов (итоговый score 0–100):

| Фактор | Вес |
|---|---|
| Снег vs историческое среднее | 40% |
| Скорость таяния vs порог | 25% |
| Прогноз температуры | 20% |
| Текущее затопление vs исторический максимум | 15% |

Градации: `low` < 30, `medium` 30–70, `high` > 70.

## Источники данных

| Сервис | Лицензия | Лимиты |
|---|---|---|
| Sentinel-2 (AWS S3) | Copernicus — свободно | Нет, CORS открыт |
| Element84 STAC API | Публичный | Нет явных лимитов |
| Open-Meteo | CC BY 4.0 | 10 000 req/день бесплатно |
| OpenStreetMap тайлы | ODbL | Только для прототипов, для prod нужен платный провайдер |
